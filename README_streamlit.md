# Réplica de Indicadores Públicos do Brasil - Streamlit

## Estrutura
- `app_streamlit.py`
- `replica_indicadores_publicos_br.py`
- `requirements.txt`

## Como rodar
```bash
streamlit run app_streamlit.py
```

## Observações
- O arquivo `replica_indicadores_publicos_br.py` deve ficar na mesma pasta do app.
- O upload do RMD é opcional.
- O app gera e permite baixar um arquivo Excel consolidado.
